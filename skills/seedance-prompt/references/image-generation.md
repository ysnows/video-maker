# 图片生成提示词规范

> 为角色参考图和首帧图生成可直接使用的中文图片提示词。支持多平台（NanoBanana / 即梦图片生成），默认 NanoBanana。

---

## 平台差异

| | NanoBanana (Lovart) | 即梦图片生成 |
|--|---------------------|-------------|
| 语言 | 中文 | 中文 |
| 语法 | 自然语言描述，不需要 MJ/SD 参数 | 自然语言描述 |
| 写实人脸 | 可以 | 不可（平台拦截） |
| 画质标签 | 用中文描述画面质感（"超精细画面，毛孔可见的皮肤质感"），不堆砌英文画质标签 | 可用"8K，超精细，角色设定图" |
| 画幅比 | prompt 开头写明（如"9:16竖版构图"） | prompt 开头写明 |

**通用禁止项（所有平台）：**
- ❌ Midjourney 语法（`--ar 3:4 --v 6 --style raw`）
- ❌ SD/NovelAI 语法（`masterpiece, best quality, 1girl`）
- ❌ 堆砌英文标签（`8K, masterpiece, best quality, highly detailed`）
- ✅ 用自然中文描述画面内容和质感要求

---

## 两类图片

### 1. 角色参考图（Character Sheet）

用途：角色一致性锚点，供后续视频生成中 @引用。

**画幅比**：统一 **9:16 竖版**

**prompt 结构（固定顺序，一段连续文本，句号分隔）：**

```
9:16竖版构图。[风格锚定]角色设定图，[构图视角(正面/3-4侧面/侧面)]，[景别(半身/全身)]，[背景(纯色/简洁)]，角色居中。
[面部：脸型+眼型+眼神+眉形+鼻型+唇形唇色+肤色肤质+表情神态]。
[发型：发色+长度+造型+特殊状态]。
[头饰/装饰：类型+材质+位置+附属元素]。
[服装从上到下：逐件描述领口开合/材质/颜色/新旧状态/污渍痕迹/布料物理状态/层叠]。
[道具：外观+材质+持握方式]。
[体态气质：身材体型+姿态+气质方向词]。
[光影：主光+补光+色调]。
[画质要求]。
禁止：任何文字、字幕、LOGO、水印、多余背景元素
```

**风格锚定选择（根据视频 realism_level）：**

| realism_level | 风格锚定词 |
|---------------|-----------|
| photorealistic | "写实人像摄影风格角色设定图，自然皮肤瑕疵可见" |
| semi-realistic | "半写实风格角色设定图" |
| 3d-render (超写实) | "超写实CG角色设定图，毛孔级细节" |
| 3d-render (游戏) | "UE5超写实3D渲染风格角色设定图" |
| anime | 对应的动画渲染风格角色设定图 |

**关键原则：**
- 五官必须与视频中的人种/地域特征一致（东方题材 → "东方面孔""中式五官"）
- 服装描述必须忠实还原实际质感（破旧的写破旧，崭新的写崭新），严禁美化
- 宁可少写一个细节，也不要编造一个错误的细节

**常见错误（必须避免）：**
- ❌ 唇色随意降级（原描述"深红唇"但写成"淡粉色"）
- ❌ 服装开合错误（原描述"露背"但写成"高领包裹"）
- ❌ 服饰质感偏移（原视频破旧陈旧，但写成崭新精美无瑕）
- ❌ 忽略污渍和磨损细节
- ❌ 美化倾向（将粗糙/残破不自觉地美化为精致/华丽）
- ❌ 遗漏头饰装饰品（很多角色最大的辨识度就在头饰/犄角/面纱）
- ❌ 编造不存在的细节

### 2. 首帧图（First Frame）

用途：关键镜头的视觉起点，用于即梦视频生成的首帧引导。

**画幅比**：必须与原视频一致（16:9 / 9:16 / 1:1 等）

**需要首帧图的镜头：**
- narrative_function 为 hook 的镜头（开场第一印象）
- narrative_function 为 payoff/twist 的镜头（视觉转折点）
- 包含新场景/新环境首次出现的镜头
- 包含复杂构图、多角色互动、大场景的镜头
- attention_level >= 4 的镜头

**prompt 结构：**

```
[画幅比，如"1:1方形构图"]。[风格锚定]，[画面构图：景别+角色位置+视觉重心]。
[角色状态：引用角色参考图中的关键辨识特征+当前姿态/表情]。
[环境细节：具体空间描述，材质、光影、氛围物件]。
[光影层次：主光源方向+补光+轮廓光+色温]。
[色调氛围：整体色调方案+后期风格]。
[画质要求]。
禁止：任何文字、字幕、LOGO、水印
```

## 示例

### 角色参考图示例（NanoBanana）

```
9:16竖版构图。写实人像摄影风格角色设定图，正面3/4侧面近景半身构图，纯深黑色背景，角色居中。面部：鹅蛋脸线条清秀，细长双眼皮眼型眼尾微翘，平直细眉，小巧挺鼻，裸色薄唇，偏白肤色细腻带有真实毛孔质感，眼神高冷不羁平视前方，东方面孔。发型：棕色长直发自然披散至背部。头饰：亮黄色帆布棒球帽正面佩戴。服装：宽松白色纯棉圆领短袖T恤，干净无污损，自然垂坠，内搭亮黄色紧身超短热裤，干净。道具：无。体态气质：身材高挑纤细，双臂自然微曲张开，气质方向——街头潮酷不羁。光影：顶部冷色聚光灯垂直打下，面部有轻微顶光阴影，边缘微弱轮廓光，超精细写实人像质感，皮肤毛孔可见。禁止：任何文字、字幕、LOGO、水印、多余背景元素
```

### 首帧图示例（NanoBanana，1:1 方形）

```
1:1方形构图。写实摄影风格电影级构图，大全景，人物居画面正中偏下，上方留大面积工业空间。高冷少女穿亮黄色T恤站在巨大的环形混凝土室内滑板场中心，身材高挑纤细，微微仰头。周围悬浮着大团亮黄色黏稠液体呈螺旋状环绕飞溅。环境：粗野主义混凝土墙面带有涂鸦，后方巨大黑色音响矩阵，地面有反光水渍。光影：顶部冷色聚光灯垂直打下，色温5600K，人物面部有顶光阴影。色调：极高对比度的冷峻水泥灰蓝背景与高饱和亮黄色视觉中心形成强烈碰撞。超精细画面。禁止：任何文字、字幕、LOGO、水印
```

## 图片风格自动匹配

根据视频主题匹配图片生成风格：

| 主题 | 图片风格 |
|------|---------|
| 仙侠/修真 | 3D国漫渲染、中国仙侠概念设计 |
| 古风/历史 | 中国风工笔画、水墨画、古典绘画 |
| 赛博朋克/科幻 | 未来科幻写实CG、概念设计 |
| 现实/人物 | 电影摄影写实、人像摄影 |
| 动漫 | 对应风格（日漫赛璐璐、国漫3D渲染等） |
| 街头/潮流 | 写实人像摄影、街头风格广告摄影 |
| 恐怖/暗黑 | 暗调写实摄影、恐怖概念设计 |

## 质量自检 Checklist

- [ ] 角色参考图画幅比为 9:16 竖版
- [ ] 首帧图画幅比与原视频一致
- [ ] prompt 开头写明了画幅比
- [ ] 五官人种特征与视频一致（东方题材→中式五官）
- [ ] 服装新旧状态与原视频一致（不美化不丑化）
- [ ] 没有使用 MJ/SD 语法或堆砌英文标签
- [ ] 所有中文描述
- [ ] 末尾有禁止项声明
- [ ] 风格锚定与 realism_level 一致
