# 深度美学约束系统

> 在基础坐标编码（[camera-codec.md](camera-codec.md)）和通用约束之上，叠加**专业级渲染美学标准**——让每一帧都对标 3D 渲染器级别的品质。
>
> 核心理念：不是"看起来像视频"，而是"看起来像 Octane 渲出来的"。

---

## Part 1｜渲染标准：对标专业 3D 渲染器

AI 视频生成的默认输出往往带有"AI 味"——光影平淡、材质塑料感、阴影不真实。深度美学约束的目标是通过精准的提示词，将输出品质拉到 **Octane / V-Ray / Arnold** 级别。

### 光影真实性（Global Illumination）

| 约束维度 | 提示词关键词 | 效果 | 对标渲染器特性 |
|:---|:---|:---|:---|
| 全局光照 | "Global Illumination, 全局光照" | 光线在场景中真实弹射 | Octane GI |
| 次表面散射 | "Subsurface Scattering, SSS" | 皮肤/半透明材质的光线穿透感 | V-Ray SSS2 |
| 焦散 | "Caustics, 焦散光斑" | 透明物体折射产生的光斑 | Arnold Caustics |
| 体积光 | "Volumetric Lighting, 丁达尔效应" | 光线穿透尘埃/雾气的可见光柱 | Octane VDB |
| HDRI 环境光 | "HDRI lighting, 环境光映射" | 真实环境光反射在物体表面 | 通用 |
| 物理衰减 | "Inverse-square falloff, 光线物理衰减" | 距光源越远亮度按平方衰减 | 物理正确灯光 |

### 材质微观细节

| 材质 | 提示词关键词 | 微观细节 | 质量对标 |
|:---|:---|:---|:---|
| 金属 | "Metallic surface, 微划痕, anisotropic highlight" | 各向异性高光 + 微划痕 + 真实反射 | Octane Metal |
| 皮肤 | "SSS skin, 毛孔细节, micro-detail" | 毛孔 + 次表面散射 + 微血管 | V-Ray AlSurface |
| 玻璃 | "Dispersion glass, 色散, IOR 1.52" | 色散彩虹 + 焦散 + 折射率精确 | Arnold Dielectric |
| 织物 | "Fabric weave, 编织纹理, 微绒毛" | 纤维级编织 + 绒毛 + 光散射 | Octane Fabric |
| 液体 | "Meniscus, 液面张力, 液态流动" | 弯月面 + 粘度 + 飞溅物理 | Houdini FLIP |
| 塑料/树脂 | "Glossy coat, 清漆层, 微颗粒" | 多层涂层 + 底层颗粒 + 清漆反射 | V-Ray CoatMtl |

### 渲染品质约束词（直接复制追加）

**基础版（20字，日常使用）：**

```
Octane 渲染品质，全局光照，材质细节真实，物理正确光影。
```

**进阶版（50字，商业出片）：**

```
Octane render quality, Global Illumination, physically accurate lighting,
SSS skin detail, micro-surface imperfections, anisotropic highlights,
volumetric atmosphere, cinematic lens flare.
```

**极致版（80字，电影级）：**

```
对标 Octane/V-Ray 渲染品质。全局光照 GI，物理正确的光线衰减，
皮肤次表面散射，金属各向异性高光，微划痕纹理，
体积光丁达尔效应，真实焦散，色彩空间 ACEScg，
运动模糊物理准确，景深散景真实光学模拟。
```

## Part 2｜构图倾向：极简先锋 + 冷暖强对比

> 在方寸之间，用最少的元素制造最强的视觉张力。借鉴包豪斯极简主义 + 赛博朋克色彩学 + 当代先锋美术指导。

### 冷暖色调对比系统

| 色调组合名 | 冷色（暗部/背景） | 暖色（亮部/主体） | 情绪节奏 | 典型场景 |
|:---|:---|:---|:---|:---|
| **赛博霓虹** | #0A0E27 深空蓝 + #6C3EFF 电光紫 | #FF2D55 霓虹红 + #FF6B35 能量橙 | 紧张→爆发 | 科技产品、夜间动作 |
| **极简白金** | #F5F5F7 苹果白 + #E0E0E0 铂金灰 | #FFD700 金色点缀 + #1A1A1A 炭黑 | 宁静→聚焦 | 奢侈品、品牌广告 |
| **冰火对冲** | #001F3F 深海蓝 + #0074D9 钴蓝 | #FF4136 烈焰红 + #FF851B 橙黄 | 对峙→冲突 | 对决、竞技、预告片 |
| **赛博冷调** | #0D1117 GitHub 暗 + #161B22 碳灰 | #58A6FF 冰蓝 + #7EE787 矩阵绿 | 冷静→渗透 | 黑客、数据可视化 |
| **暮光渐变** | #2D1B69 暮紫 + #11998E 深青 | #FC5C7D 落霞粉 + #FFC371 暮金 | 舒缓→回味 | 情感叙事、MV |
| **极简黑白** | #000000 纯黑 | #FFFFFF 纯白 + 单一点缀色 | 克制→冲击 | 先锋艺术、时尚 |

### 极简先锋构图法则

```
1. 减法优先
   画面中的视觉元素 ≤ 3 个，多一个都是噪声。
   "如果一个元素去掉后画面没变弱，那它一开始就不该在。"

2. 强对比分割
   用色彩温度（冷/暖）分割画面的叙事节奏：
   ┌──────────┬──────────┐
   │ 冷色区域  │ 暖色区域  │
   │ (暗部)    │ (亮部)    │
   │ 紧张/等待 │ 爆发/行动 │
   └──────────┴──────────┘

3. 呼吸留白
   画面至少 30% 为负空间（留白/纯色区域）。
   留白 = 高级感。塞满 = 廉价感。

4. 几何锚点
   用一条对角线或一个几何中心锚定视觉重心。
   避免"主体居中，四面对称"的平庸构图（除非刻意韦斯·安德森风格）。

5. 层次递进
   前景(虚) → 中景(实) → 背景(虚)，至少三层景深。
   或用色彩明度：暗 → 亮 → 暗，制造视觉通道。
```

## Part 4｜美学约束提示词模板

### 模板 A：赛博朋克科技风

```
坐标编码：Z4 + Y3 + X2 + 缓慢推近 + F:50mm 浅景深

美学约束：
"Octane render quality, Global Illumination, physically accurate lighting.
 赛博朋克色调：深空蓝 (#0A0E27) 为暗部基调，霓虹红 (#FF2D55) 为高光点缀。
 材质：金属表面各向异性高光，微划痕纹理清晰可见。
 极简构图：画面元素 ≤ 3，背景 40% 负空间。
 体积光穿透大气雾气，霓虹光源边缘色散。"
```

### 模板 B：极简白金品牌风

```
坐标编码：Z3 + Y4 + X 全环绕 + F:85mm 散景

美学约束：
"V-Ray 渲染品质，柔和的三点式摄影棚布光，HDRI 环境光映射。
 极简白金色调：纯白背景 (#F5F5F7)，产品表面金色点缀 (#FFD700)。
 材质：产品表面清漆涂层清晰，微颗粒质感，底层纹理可见。
 构图：主体占画面 40%，60% 为纯净留白，减法到极致。
 散景光斑圆润真实，无杂色边缘。"
```

### 模板 C：冰火对冲动作风

```
坐标编码：Z6 + Y2 + X1 + 缓慢推近 + F:24mm 逆光

美学约束：
"Octane render quality, 大光比 Key:Fill ≥ 6:1，全局光照。
 冰火色调：深海蓝 (#001F3F) 暗部 vs 烈焰红 (#FF4136) 亮部。
 皮肤次表面散射，汗珠反射环境光。
 强对比分割构图：画面左冷右暖，冷暖交界线恰好切过角色面部。
 体积光从背后穿透烟雾，边缘光勾勒轮廓。"
```

## Part 5｜与现有史诗结构的关系

本文件是 `SKILL.md` 中「史诗/大制作结构」的深度扩展。两者关系：

| 层级 | 来源 | 作用 |
|:---|:---|:---|
| 品质锚定（开头） | SKILL.md | 声明渲染引擎 + VFX 等级 |
| 渲染标准 | 本文件 Part 1 | 告诉模型光影/材质要达到的物理精度 |
| 色调系统 | 本文件 Part 2 | 确保全片视觉风格统一 |
| 光影三层 | vocabulary.md | 光源 → 光行为 → 色调的分层描述 |
| 大气连贯声明 | SKILL.md | 统一全片物理效果 |
| 收束句 | vocabulary.md | 后期处理 + 张力宣言 |

**建议叠加顺序：**

```
[时长][品质锚定（SKILL.md）]，[色调组合（本文件 Part 2）]，
[大气连贯声明（SKILL.md）]，[渲染品质约束（本文件 Part 1）]，

0-Xs：[分镜画面] + [坐标编码（camera-codec.md）]...

光影：[光源] + [光行为] + [色调]（vocabulary.md 光影三层），
[收束句（vocabulary.md）] + [张力宣言]。
```
